# Rad Tech Tutor Funnel (Template)

This is a clean, working Next.js (App Router) + Tailwind v4 template using CSS variables as the theme source of truth.

## Run locally
```bash
npm install
npm run dev
```

Open:
- http://localhost:3000

## Theme rules
- Teal = success/correct/money
- Yellow = neutral/locked/CTA
- Red = wrong/danger
- Everything else = dark minimal

Theme variables + helpers live in:
- `app/globals.css`
